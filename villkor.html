Regler och villkor

Utkast inför lansering. Texten ska ersättas med skarp juridisk version innan publik launch.

Medlemskap

Medlemskapet ger tillgång till utvalda produkter och medlemspriser så länge prenumerationen är aktiv och betald.

Priser

Priser kan ändras över tid. Det pris som visas i checkout vid köptillfället är det pris som gäller för ordern.

Konto

Kunden ansvarar för att använda korrekt email och hålla inloggningsuppgifter skyddade. Versen kan stoppa åtkomst vid misstänkt missbruk.

Betalning

Betalningar hanteras via Shopify checkout och anslutna betalningsleverantörer. Medlemskap kan hanteras som prenumeration via ReCharge.

Missbruk

Delning av medlemsaccess, automatiserad scraping eller försök att kringgå medlemskrav kan leda till avstängt konto.
